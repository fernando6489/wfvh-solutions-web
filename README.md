# wfvh-solutions-web

# 🌐 Creación de página web – WFVH Solutions

## 📌 Visión
Plataforma web para Cine Nova, desarrollada por **WFVH Solutions**, aplicando metodología ágil XP.

## 🎯 Objetivo
Permitir a los clientes consultar cartelera, seleccionar funciones, comprar entradas y reservar asientos en línea.

## ⚙️ Artefactos XP aplicados
- Historias de usuario
- Tarjetas CRC
- Metáfora del sistema ("Taquilla virtual")
- Pruebas unitarias automatizadas
- Integración continua (CI/CD)
- Refactorización constante

## 📂 Estructura del repositorio
- `/docs` → Documentación del proyecto  
- `/src` → Código fuente  
- `/tests` → Pruebas XP  
- `.github/workflows` → Pipeline CI/CD
